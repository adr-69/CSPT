Password Cracking (Brute Force)

What is Password Cracking (Brute Force)?

Password Cracking (Brute Force) is a method used to find passwords by systematically trying every possible combination of characters until the correct one is found. It is one of the simplest yet slowest techniques for password attacks.


Why is it Important in Ethical Hacking and Penetration Testing?

In ethical hacking, brute force attacks are used to test how secure a system is against weak passwords. This method helps penetration testers identify weak login credentials and encourage the use of strong password policies. It demonstrates how vulnerable a system can be if users rely on simple, short, or commonly used passwords.

  • Tests password strength against brute-force attempts
  • Identifies weak credentials in authentication systems
  • Demonstrates need for multi-factor authentication (MFA)
  • Supports the implementation of account lockout policies

Python Code: Brute Force Password Cracker


import itertools
import string

target_password = "ab" # The password we want to guess
characters = string.ascii_lowercase + string.digits

found = False

for length in range(1, 6): # Adjust max length for deeper cracking
for guess in itertools.product(characters, repeat=length):
attempt = ''.join(guess)
print(f"Trying: {attempt}")
if attempt == target_password:
print(f"[+] Password found: {attempt}")
found = True
break
if found:
break

if not found:
print("[-] Password not cracked.")
