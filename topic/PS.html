Port Scanner

What is a Port Scanner?

A Port Scanner is a tool used in ethical hacking and penetration testing to detect which ports are open on a target machine or server. These open ports may indicate potential entry points if services running behind them are vulnerable.


Why is a Port Scanner Important?

Port scanning helps security professionals:

  • Identify exposed services (e.g., SSH, FTP, HTTP)
  • Detect unnecessary or misconfigured open ports
  • Assess the attack surface before deeper testing
  • Understand which services may be vulnerable

Knowing which ports are open is a critical first step before vulnerability scanning or exploit development.

Python Code: Port Scanner


import socket

# CHANGE IP TO YOUR TARGET
target = "192.168.8.1" #Here
start_port = 1
end_port = 100 #SCAN OPEN PORTS 1 - 100

print(f"\nScanning {target} from port {start_port} to {end_port}...\n")

found_open = False

for port in range(start_port, end_port + 1):
sock = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
sock.settimeout(0.5)

result = sock.connect_ex((target, port))
if result == 0:
print(f"[+] Port {port} is open")
found_open = True
sock.close()

if not found_open:
print("No open ports found.")
