Network Discovery & Mapping

What is Network Discovery & Mapping?

Network Discovery & Mapping is the process of identifying all active devices, hosts, and systems within a specific network. It involves scanning a network range to gather details such as hostnames, open ports, and running services.


Why is it Important in Ethical Hacking & Penetration Testing?

In ethical hacking and penetration testing, network discovery is a crucial first step in the reconnaissance phase. Before performing exploits or vulnerability assessments, a penetration tester must know:

  • What devices are active on the network
  • What IP addresses are being used
  • What kind of systems are present (e.g., servers, printers, IoT devices)
  • How these systems are structured or interconnected

This information helps ethical hackers plan their next steps and prioritize targets for further analysis or exploitation. It's like creating a digital map of the battlefield before making any move.

Python Code: Network Discovery & Mapping



import ipaddress
import subprocess
import socket
from concurrent.futures import ThreadPoolExecutor

# CHANGE IP YOU WANT TO PING
network = ipaddress.ip_network('192.168.1.1/24', strict=False)

def ping(ip):
try:
output = subprocess.check_output(['ping', '-c', '1', '-W', '1', str(ip)],
stderr=subprocess.DEVNULL)
try:
hostname = socket.gethostbyaddr(str(ip))[0]
except socket.herror:
hostname = "Unknown"
print(f"[+] Active: {ip} ({hostname})")
except subprocess.CalledProcessError:
pass
with ThreadPoolExecutor(max_workers=100) as executor:
executor.map(ping, network.hosts())
